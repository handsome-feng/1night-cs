# 计算机组成与设计

## 计算机的语言：MIPS

寄存器

$s0 - $s7:   16 - 23; 变量对应的寄存器

$t0 - $t7:    8 - 15; 临时寄存器

$zero:    恒为0

$a0 - $a3

$v0 - $v1

$gp

$fp

$sp

$ra

$at:	    保留，用于处理大的常数



32个的理由：

1. 越小越快
2. 指令格式位数的限制



指令：

add    $s1, $s2, $s3    # $s1 = $s2 + $s3

sub    $s1, $s2, $s3    # $s1 = $s2 - $s3

addi   $s1, $s2, 20      # $s1 = $s2 + 20



补码取反加1原理：
$$
x + \bar{x} = -1 \\
x + \bar{x} + 1 = 0 \\
-x = \bar{x} + 1
$$
