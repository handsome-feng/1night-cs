msginit -l bo  //根据当前目录下pot文件，生成bo.po

```shell
for po in `ls *.po`; do msgmerge -U $po slideshow-ubuntukylin.pot; done
```

修改po文件：

1. 在Linux世界里有软件可以直接打开.mo文件以供编辑，这款软件叫——Virtaal

2. ```
   msgunfmt foo.mo -o foo.po
   // Edit foo.po
   msgfmt foo.po -o foo.mo
   ```
